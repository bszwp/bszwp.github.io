### 1.安装并检测bower

> npm install bower -g
>
> bower -v

### 2.初始化bower仓库

> bower init
>
> 将在本目录下生成bower_components文件夹，并生成**bower.json**文件

### 4.安装指定包

> bower install packageName --save
>
> 安装指定包并添加到bower.json文件
